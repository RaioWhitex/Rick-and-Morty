# 🛸 Rick and Morty - Personagens

Uma aplicação que consome a API de Rick and Morty para exibir todos os personagens da série em cards, com tema escuro, layout em grade e interface responsiva.

Ideal para praticar consumo de APIs e exibição dinâmica de dados.

## ✨ Tecnologias Utilizadas

- HTML5
- CSS3
- JavaScript

## 🎨 Características do Design

- Visual moderno com tema escuro
- Seção de destaque (hero) com imagem e chamada
- Cards de personagens em grade
- Indicadores de status com cores
- Layout responsivo

## 🚀 Objetivo do Projeto

Praticar o consumo da API de Rick and Morty, explorando requisições, manipulação de dados em JavaScript e renderização dinâmica de cards, além da estilização com CSS.

## 📚 Funcionalidades

- Listagem de todos os personagens
- Exibição de imagem, nome e informações de cada personagem
- Indicador de status (vivo, morto, desconhecido)
- Dados de espécie e localização
- Interface leve e responsiva

## 💼 Possíveis Aplicações

- Galeria de personagens
- Catálogo baseado em API
- Projeto de estudo de consumo de APIs
- Componente para sites de fãs
- Base para sistemas de busca e filtros
